You will need

  • Raspberry Pi, power supply, case
  • 64GB micro SD Card
  • Ethernet Cable

Instructions

Download this image

Follow the to write the image to your SD card

Linux Instructions
Windows Disk imager

Eject and remove your SD card and assembly your Pi with the new card

Start your Raspberry Pi

Boot up and login with
username eth
password eth

Change the password with the passwd command

Resize the partition to use the whole SD card

type sudo fdisk /dev/mmcblk0

re create the partition by entering the following when prompted

d – delete
2 – partition to be deleted
n – create new partition
p – select partition type primary
enter - select default
enter – select default start block
enter – select default last block
w – to write our changes

Now reboot the machine :
sudo reboot

when the machine restarts login again
then run

sudo resize2fs /dev/mmcblk0p2 – to expand the partition

You can check with df -h
