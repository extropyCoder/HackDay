Starting Geth

Macosx

geth --datadir data --networkid 20161105 --ipcpath ~/Library/Ethereum/geth.ipc console

Linux

geth --datadir data --networkid 20161105 --ipcpath ~/.ethereum/geth.ipc console

Windows

From the !!! directory run the following from a command prompt geth --datadir data --networkid 20161105 console

Stopping Geth

From the console type exit
