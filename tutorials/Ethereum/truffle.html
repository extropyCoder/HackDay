We have used a local geth node and the online solidity compiler / ethereum wallet We will now move to use the truffle IDE and testrpc to give us a local test node You need to install nodejs https://nodejs.org/en/download/ https://github.com/ConsenSys/truffle http://truffle.readthedocs.io/en/latest/getting_started/installation/ to install do npm install -g truffle Documentation http://truffle.readthedocs.io/en/latest/ Also install testrpc https://github.com/ethereumjs/testrpc TO create a project in an empty directory type truffle init this will create a dummy project for a crypto currency the directory structure is as follows : project \app \contracts \test
